# 日本式データ印画像ジェネレーター

日本でよく使われるデータ印（回転式日付印）風の印影を、ブラウザだけで作成・出力できるSPAです。

## 主な機能
- 上枠/日付/下枠テキストの編集
- 日付フォーマット切替（和暦 / 西暦）と区切り文字指定
- 書体切替（明朝体 / ゴシック体）
- テキスト色・枠線色・枠線太さの調整
- PNG / SVG ダウンロード（PNGは透過背景対応）
- LocalStorage履歴（最新10件）からの編集再開

## 使い方
1. 入力設定でテキスト・日付・書体・色・線幅を調整します。
2. プレビューで見た目を確認します。
3. 必要に応じて「現在の設定を履歴に保存」を押し、再利用できるようにします。
4. PNG または SVG ダウンロードボタンで画像を保存します。

## セットアップ

```bash
npm ci
npm run dev
```

```bash
npm run test
npm run lint
npm run build
```

## 動作要件（非機能）
- モダンブラウザ（最新の Chrome / Edge / Firefox / Safari）を対象とします。
- すべての処理はクライアントサイドで完結し、サーバー通信は行いません。
- 入力データは外部送信せず、履歴はブラウザのLocalStorageのみに保存します。
- 通常操作時は入力変更に対して体感的に遅延なく再描画されることを目標とします。
